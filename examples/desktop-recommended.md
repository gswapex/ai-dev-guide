# 1. 项目概述

## 1.1 项目名称
个人备忘录工具

## 1.2 项目一句话定义
一个运行在 Windows 桌面上、支持新增、编辑和删除备忘记录的本地工具软件。

## 1.3 项目类型
Windows 桌面端应用 / 工具类

## 1.4 目标平台
Windows 桌面端（PC）

## 1.5 当前阶段
第一版 Demo 开发阶段

## 1.6 本次任务目标
先完成备忘录的增删改查功能，数据保存到本地文件

# 2. 项目目标与成功标准

## 2.1 项目目标
完成一个可运行的备忘录桌面工具，支持本地数据持久化

## 2.2 用户价值
- 目标用户是：需要在电脑上记录待办事项的个人用户
- 他们当前的主要问题是：便签纸容易丢，电脑记事本没有分类管理
- 核心价值：本地化备忘管理，关闭后数据不丢失

## 2.3 成功标准
- 可以新增备忘记录
- 可以编辑已有记录
- 可以删除记录
- 关闭重开后数据仍然存在

## 2.4 演示标准
- 程序可以启动运行
- 数据保存到本地 JSON 文件
- 可以演示增删改查完整流程

# 4. 核心问题与使用场景

## 4.1 当前问题
- 重要的事情记在便签纸上容易丢失
- 记事本没有分类，找起来麻烦

## 4.2 目标场景
- 用户想到一件事时快速打开软件记录
- 查看今天需要做的事情
- 完成后删除或标记

## 4.4 最关键主流程
用户打开软件 → 看到备忘列表
→ 点击「新增」→ 填写标题和内容 → 保存
→ 记录出现在列表中
→ 关闭重开后记录依然存在

# 7. 数据结构定义

## 7.1 实体列表
- Note（备忘记录）

## 7.2 字段定义

### 实体：Note（存储为 JSON 数组）

| 字段名 | 类型 | 是否必填 | 含义 | 示例值 |
|---|---|---|---|---|
| id | string | 是 | 唯一ID（时间戳） | "1710000001" |
| title | string | 是 | 备忘标题 | "买东西" |
| content | string | 否 | 备忘内容 | "牛奶、面包、鸡蛋" |
| createdAt | string | 是 | 创建时间 | "2024-03-10 10:00" |
| updatedAt | string | 是 | 最后修改时间 | "2024-03-10 14:00" |

## 7.3 本地存储说明
- 数据保存为 `notes.json` 文件
- 文件路径：程序运行目录下
- 每次增删改后重写整个文件

---

# 8. 业务规则

## 8.1 数据规则
- 标题不能为空
- 内容可以为空
- 列表按最后修改时间倒序排列

## 8.2 操作规则
- 点击列表项，右侧编辑区显示该记录内容
- 修改后点击保存才生效，不自动保存
- 删除操作需要二次确认弹窗

---

# 9. 界面与交互要求

## 9.1 整体风格
简洁的左右分栏布局，类似系统自带便签风格

## 9.2 布局要求
- 左侧列表宽度约 200px，右侧编辑区占剩余空间
- 当前选中记录在列表中高亮显示

## 9.3 反馈要求
- 保存成功后状态栏显示「已保存」
- 删除前弹出确认对话框
