## 一、用户管理

### 1.1 界面：用户登录 / 登出

| 交互元素 | 类型 | 说明 |
|---|---|---|
| 登录 | 按钮 | 提交登录请求 |
| 登出 | 按钮 | 退出当前登录 |
| 用户名 | 输入框 | 输入账号用户名 |
| 密码 | 输入框 | 输入登录密码 |
| 记住登录状态 | 复选框 | 保持登录状态 |

### 1.2 界面：用户注册

| 交互元素 | 类型 | 说明 |
|---|---|---|
| 注册 | 按钮 | 提交注册信息 |
| 用户名 | 输入框 | 设置账号用户名 |
| 邮箱 | 输入框 | 输入注册邮箱 |
| 密码 | 输入框 | 设置登录密码 |
| 密码强度提示 | 标签 | 实时显示密码强度 |

### 1.3 界面：角色管理

| 交互元素 | 类型 | 说明 |
|---|---|---|
| 新增角色 | 按钮 | 创建新角色 |
| 编辑角色 | 按钮 | 修改角色信息 |
| 删除角色 | 按钮 | 删除已有角色 |
| 选择权限 | 下拉框 | 为角色选择对应权限 |

### 1.4 界面：权限控制

| 交互元素 | 类型 | 说明 |
|---|---|---|
| 保存权限设置 | 按钮 | 保存权限配置 |
| 权限项 | 勾选框 | 勾选/取消具体权限项 |
| 权限审计日志 | 标签 | 查看权限操作记录 |

### 1.5 界面：密码修改

| 交互元素 | 类型 | 说明 |
|---|---|---|
| 修改密码 | 按钮 | 提交密码修改 |
| 原密码 | 输入框 | 输入当前密码 |
| 新密码 | 输入框 | 输入新密码 |
| 确认新密码 | 输入框 | 再次确认新密码 |
