description: 生成项目文档并进入开发
---

Load and use the installed `ai-dev-guide` skill.

In Chinese, initialize the current project for development:

- verify the confirmed project information is sufficient
- generate `docs/project.md`
- generate `docs/structure.md`
- if it is a Spring Boot project, also generate `docs/architecture.md`
- generate `docs/tasks.md`
- recommend the first task instead of letting the user choose arbitrarily

Do not start coding before the documents are generated.

Follow the installed skill's `SKILL.md`, `AGENTS.md`, and `references/templates.md`.
